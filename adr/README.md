# Architecture Decision Records (ADR)

Ce dossier contient les enregistrements des décisions d'architecture (ADR) pour le projet DEX Pair Explorer.

## Format

Chaque ADR suit le format suivant:
- **Titre**: Décision prise
- **Statut**: Accepté, Proposé, Déprécié, Remplacé
- **Contexte**: Situation et problème à résoudre
- **Décision**: Solution choisie
- **Conséquences**: Impacts positifs et négatifs
